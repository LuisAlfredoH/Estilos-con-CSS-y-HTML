Descripción

Hola de nuevo, este es el primer tutorial en el que les enseñare a como convertir GitHub en un servidor web estático. Con estático me refiero a que este servidor no podrá almacenar ni procesar datos.

En conclusión, convertiremos un repositorio de github en un servidor web estático en el cual podrán ejecutarse archivos .html, .css, .js, .jpg, .gif, .png, .mp3 y .mp4. No podrán subir archivos .php, .asp, .jsp, .py ni .rb porque el servidor no los ejecutara.

Si tienes alguna duda, no dudes en informarlo en la caja de comentarios que esta abajo. Disfruten del tutorial y nos vemos hasta la proxima
